# Great Reed Warbler

## Description

Data and analyses of Great Reed Warbler tracks from European populations in Bulgaria (Kalimok 2015-2017), Czechia (Mutenice 2016-2018) and Russia (Rybachy 2016-2017).

This data was collected by the Swiss Ornithological Institute.
